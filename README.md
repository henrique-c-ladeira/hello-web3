# Hello Web3

Sample web3 app using wagmi

## Requirements

- node
- pnpm `npm install --global pnpm`

## Run

`git clone`

`cd hello-web3`

`pnpm install`

`pnpm run dev`

Connect an wallet in the browser using the application buttons.
